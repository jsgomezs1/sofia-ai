# Product Overview

LiveKit Meet is an open-source video conferencing application that provides real-time audio and video communication capabilities. Built on LiveKit's WebRTC infrastructure, it serves as both a production-ready conferencing solution and a reference implementation for developers building with LiveKit.

## Key Features

- Real-time video and audio conferencing
- Screen sharing and recording capabilities
- End-to-end encryption (E2EE) support
- Noise filtering with Krisp integration
- Camera and microphone settings management
- Keyboard shortcuts for common actions
- Performance optimization for various network conditions
- Customizable UI themes

## Target Users

- Teams needing a self-hosted video conferencing solution
- Developers learning to build with LiveKit Components
- Organizations requiring open-source communication tools
