# Theme Test Page

This page is used to verify that the shadcn/ui theme configuration is working correctly with the purple accent color scheme.

## Purpose

- Verify all required shadcn/ui components are installed and working
- Test the purple accent color scheme (#DEC9F2, #E1D5F2)
- Test the light gray background (#F2F2F2) and text color (#595959)
- Ensure all component variants render correctly

## Components Tested

- Button (all variants)
- Card
- Input
- Select
- Checkbox
- Tabs
- Badge
- Dialog
- Label
- Form elements

## Access

Visit `/theme-test` to view the test page.

## Theme Colors

- Background: #F2F2F2 (light gray)
- Foreground: #595959 (gray text)
- Primary/Accent: #DEC9F2 (purple)
- Accent Foreground: #E1D5F2 (light purple)

This test page can be removed after the UI redesign is complete and verified.
